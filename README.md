# 🚀 Auto-Publish - 微信公众号+小红书全自动内容发布系统

18天日更验证的完整流水线：选题 → AI写作 → 微信排版 → 小红书卡片 → 定时发布

## 📦 项目结构

```
├── vercel-api/          # Vercel API服务（5个在线API）
│   ├── api/             # 文本人性化、标题生成等
│   └── cloud-functions/ # 腾讯云函数部署版
├── xhs_cards/           # 小红书手绘风卡片素材
├── scripts/             # 自动化脚本（UUMit巡航等）
└── docs/                # 使用文档
```

## ⚡ 核心功能

| 模块 | 功能 | 状态 |
|------|------|------|
| WeWrite | 选题→写作→排版→草稿箱全流程 | 🟢 18天验证 |
| gen_xhs | 小红书手绘风卡片一键生成 | 🟢 日更中 |
| Vercel API | 文本人性化/标题生成/内容检测 | 🟢 5个在线 |
| 内容合规 | 自动扫禁词/降风险 | 🟢 合规稳定 |

## 🎯 适用人群

- 公众号日更创作者
- 小红书内容运营
- 自媒体矩阵管理
- 想用AI提效10x的内容团队
