## 一、京东联盟 完整跑通路线

### Step 1：获取 AppKey + Secret（你现在在浏览器做）

1. 打开 https://union.jd.com/
2. 用你的京东账号登录（已经有的话）
3. 点「推广管理」→「推广工具」→「创建应用」
4. 选择类型：**「导购媒体」**（适合公众号/社群推广）
5. 填写：
   - **应用名称**：公众号好物推荐
   - **应用描述**：通过微信公众号内容推荐京东高佣商品，为粉丝提供购物参考
   - **网站地址**：https://github.com/htywba-eng
   - **回调地址**：https://vercel-api-45446.vercel.app/api/jd-callback
6. 创建后，在「应用管理」里找到 AppKey 和 AppSecret
7. 把这两个发给我（或让我通过浏览器看）

⚠️ **AppSecret 只显示一次**，立即复制保存！

### Step 2：部署真正的 JD 搜品 API

拿到 Key 后，我写一个真正能跑的 Vercel API：

```
POST /api/jd-search
入参：keyword（搜索词）, sort（排序：commission/price/sales）
返回：商品列表，每条含：
  - 标题、图片、到手价
  - 佣金比例、预估佣金（元）
  - 优惠券金额
  - 推广链接（union-click.jd.com）
  - 30天引单量
```

### Step 3：自动生成带货推文

```
输入：品类关键词（如"纸巾"、"洗发水"）
流程：
  1. 调 JD API 搜该品类高佣商品（佣金排序取前5）
  2. Claude 写推荐文案（合规版，不吹收入）
  3. 嵌入推广链接
  4. 输出公众号格式 → 一键推到草稿箱
```

## 二、拼多多多多进宝 路线

### Step 1：确认审核状态

1. 打开 https://open.pinduoduo.com/
2. 登录你之前提交的账号
3. 进「控制台」→「我的应用」→ 看审核状态
4. 如果还在审核中或驳回：
   - 驳回原因一般是 PRD 文档不完整 → 我帮你写一份
   - 重新提交，通常24小时内出结果

### Step 2：审核通过后的配置

和 JD 同理：
- 获取 client_id + client_secret
- 绑定多多进宝（jinbao.pinduoduo.com）获取 PID
- 部署 Vercel API：POST /api/pdd-search
- 接入推文自动生成

### Step 3：JD vs PDD 策略差异

| | 京东 | 拼多多 |
|------|------|------|
| 客单价 | 高（50-200元） | 低（9.9-50元） |
| 佣金率 | 1-5% | 5-30% |
| 适合品类 | 家电/数码/日百 | 日用品/零食/家居 |
| 文案策略 | 品质推荐 | 性价比/避坑指南 |
| 目标读者 | 公众号粉丝（中产） | 公众号+社群 |
